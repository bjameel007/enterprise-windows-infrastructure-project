# Enterprise Windows Infrastructure Project

> A Windows Server 2022 infrastructure project demonstrating enterprise systems administration using a World Cup-themed business scenario.

## Overview

This project was built to strengthen practical Windows Systems Administration skills by designing, deploying, automating, and managing a complete Active Directory environment from the ground up.

Rather than following isolated tutorials, the objective was to simulate how enterprise administrators manage users, security, file services, and Group Policy in a real-world environment.

---

## Technologies

- Windows Server 2022
- Windows 11
- Active Directory Domain Services (AD DS)
- PowerShell
- CSV Automation
- Group Policy
- SMB File Shares
- NTFS Permissions
- Security Groups (RBAC)

---

## Project Objectives

- Design a scalable Active Directory structure
- Automate user provisioning with PowerShell
- Implement Role-Based Access Control (RBAC)
- Configure SMB file shares
- Apply NTFS permissions
- Deploy Group Policy Objects (GPO)
- Validate end-user access
- Troubleshoot common enterprise administration issues
